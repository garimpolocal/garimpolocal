**Plataforma open-source que conecta residentes de Angola a trabalhadores locais qualificados e empresas em todas as 21 províncias.**

## Sobre o Projeto

O Garimpo Local é uma plataforma inovadora desenvolvida para fortalecer a economia angolana, facilitando a conexão entre:
- **Clientes** que precisam de serviços profissionais
- **Garimpeiros** (trabalhadores locais) qualificados que oferecem seus serviços
- **Empresas** que buscam talentos e publicam projectos em grande escala

## Funcionalidades Principais

### Para Clientes
- Publicação de pedidos de serviços
- Busca de profissionais por categoria, localização e avaliações
- Comunicação segura via WhatsApp
- Sistema de avaliação e feedback
- Cobertura em todas as 21 províncias de Angola

### Para Garimpeiros
- Perfil profissional verificável
- Portfolio de serviços e histórico
- Acesso a oportunidades de trabalho
- Sistema de ranking e reputação
- Dashboard para gestão de serviços

### Para Empresas
- Cadastro empresarial completo
- Gestão de projectos em grande escala
- Acesso a talentos qualificados
- Publicação de vagas e projectos
- Suporte dedicado via WhatsApp

## Stack Tecnológica

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Node.js com tRPC
- **Database**: PostgreSQL com Drizzle ORM
- **Hosting**: Firebase Hosting
- **Authentication**: Firebase Auth
- **Real-time**: Firebase Firestore
- **Storage**: Firebase Storage

## Como Começar

### Pré-requisitos
- Node.js (v18 ou superior)
- npm ou yarn
- Conta Firebase

### Instalação

```bash
# Clone o repositório
git clone https://github.com/garimpolocal/garimpolocal.git
cd garimpolocal

# Instale as dependências
npm install

# Configure as variáveis de ambiente
cp .env.example .env

# Inicie o servidor de desenvolvimento
npm run dev
```

### Configuração Firebase

1. Crie um projeto no [Firebase Console](https://console.firebase.google.com/)
2. Adicione uma aplicação web
3. Copie as configurações para o arquivo `firebase-config.js`
4. Configure Firestore e Authentication

## Cobertura Geográfica

O Garimpo Local atende todas as **21 províncias de Angola**:

Bengo, Benguela, Bié, Cabinda, Cuando, Cubango, Cuanza Norte, Cuanza Sul, Cunene, Huambo, Huíla, Icolo e Bengo, Luanda, Lunda Norte, Lunda Sul, Malanje, Moxico, Moxico Leste, Namibe, Uíge, Zaire
